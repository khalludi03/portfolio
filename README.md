# Khaled Kashem — Portfolio

Personal portfolio website built with plain HTML, CSS, and vanilla JavaScript.

**Live site:** https://khalludi03.github.io/portfolio

## Sections
- Hero with typewriter animation
- About & Education
- Technical Skills
- Projects
- Experience
- Achievements (with animated counters + CP platform ratings)
- Contact

## Setup (GitHub Pages)
1. Push this repo to GitHub as `portfolio`
2. Go to **Settings → Pages**
3. Source: **Deploy from branch** → `main` → `/ (root)`
4. Save — site goes live at `https://<username>.github.io/portfolio`
